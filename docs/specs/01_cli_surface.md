# CLI Surface

The active operator interface is the `moltbox` command.

Canonical domains:

- `runtime`
- `host`
- `tools`

Canonical forms:

- `moltbox runtime <environment> <verb>`
- `moltbox host <service> <verb>`
- `moltbox tools <verb>`

The CLI implementation lives under `moltbox-cli/`. The folder name is not part of the command grammar.
