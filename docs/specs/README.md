# Specifications

The active spec set is intentionally short and structural.

- `01_cli_surface.md` defines the CLI surface and domain split.
- `02_container_assets.md` defines the appliance container asset taxonomy.
- `03_runtime_stack.md` defines runtime responsibilities.
- `04_tools_lifecycle.md` defines the tooling domain lifecycle.
- `05_deployment_pipeline.md` defines deploy and rollback expectations.
