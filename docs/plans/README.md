# Plans

Use this directory for gateway-local implementation plans and test plans.

Typical contents:

- implementation plans that are specific enough to execute without reopening major design decisions
- test plans that define parser, handler, integration, and appliance validation coverage

Plans should reference the review or decision record that established the target direction.
