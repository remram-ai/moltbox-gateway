# moltbox-cli

`moltbox-cli/` is the software project that implements the `moltbox` command.

Important rule:

- The CLI command is `moltbox`.
- The folder name `moltbox-cli` is not part of the CLI namespace.

Domain folders:

- `runtime/` for runtime environment commands, diagnostics, and tests
- `host/` for appliance host-service commands, diagnostics, and tests
- `tools/` for the singleton tooling domain and the shared Python package source
